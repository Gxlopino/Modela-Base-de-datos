Pasos para correr correctamente la Base de datos.
1) Ejercutar solo las secuencias ubicada en la consulta Creacion de datos
2) Ejecutar TI que es la creacion de las tabals de la Base de Datos
3) Ejercutar por completo la consulta Creacion de datos 
4) Ejercutar la consulta de insertar datos por completo
5) Ejecutar Consultas con objetos para verificar los objetos creador, una vez ya teniendo datos del paso anterior
6) Probar la base de datos ahceindo cualquier tipo de consulta.
